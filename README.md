# BeanPkgs

This is a private arch linux pkgbuild repo which include some crack packages and some self-using softwares.

> [!Note]
> Some packages are merged from [beanc904/DevelopingHandbook](https://github.com/beanc904/DevelopingHandbook).
> So, you can find the packages' readme and release in it.

## Usage

Please add the configuration in your `/etc/paru.conf`.

```conf
[beanpkgs]
Url = https://github.com/beanc904/beanpkgs.git
Depth = 5
#SkipReview
```

## The end

Because the pkgs are summarized from my daily using and the architecture of my laptop is amd64. So, there are only x86_64 in this repo.
